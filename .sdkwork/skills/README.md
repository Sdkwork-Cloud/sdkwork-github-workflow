# SDKWork Repository Skills

Repository-local skills may be added under `.sdkwork/skills/<skill-name>/SKILL.md`.

Rules:

- Skill directory names use lowercase kebab-case.
- A real skill must have `SKILL.md` as its entrypoint.
- Skills must cite the relevant root standards instead of copying them.
- Skills must not contain source credentials, runtime data, generated SDK output, or user-private files.

See `../../AGENTS.md` and `../../../sdkwork-specs/SDKWORK_WORKSPACE_SPEC.md` before adding a skill.
